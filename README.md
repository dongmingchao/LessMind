# Less Mind 简记 APP

[![Build Status](https://travis-ci.org/dongmingchao/LessMind.svg?branch=master)](https://travis-ci.org/dongmingchao/LessMind)

一个Vue + Koa的跨平台应用，其目的是打造一款手势友好，快捷方便的记事备忘App

## Features

- [x] 云端同步
- [x] 滑动删除，拖动排序
- [x] 点击以详细设定提醒，以及详细内容
- [ ] 子步骤任务
- [ ] 支持扩展插件API
- [ ] 全自动更新，无需用户下载安装
- [ ] 云端数据全加密，保护您的隐私
- [ ] 智能清单，帮您梳理您的生活

*未打勾表示将在以后逐步更新中上线*
